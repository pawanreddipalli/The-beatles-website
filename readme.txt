* The website is for one of the greatest music bands in history called 'The beatles'. 
* There are mainly three tabs about, home, store.
* The about page contanins information about the band and their achievements.
* The home page contains all the albums by the band and when we click on view album it redirects the user to
  the spotify page of the album.
* The store page contains all the albums, merchandise that can be bought from the website. The page also shows
  details about the cart and the user can remove the contents of the cart using the remove button. User can also
  modify the quantity of the cart and see the price change realtime.
* The store.js is used to update the cart in realtime like display total according to the quantity, remove items 
  and add items to the cart.
